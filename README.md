# Ejecutar en desarrollo

1. Clonar el repositorio.
2. Ejecutar:
```
yarn install
```
3. Instalar NestJs CLI:
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos:
````
docker-compose up -d
````
